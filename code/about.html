This website displays crime statistics of different cities within close proximity of a searched city. The number of crimes reported is for the whole year of 2008. This information appears on a map. "Crime Index" is a number calculated with the following fomula:

(total number of crimes) / (population) x 100

Please do not rank, compare, or judge cities based solely on the data available on this website. There are several reasons why, which you can read about here.

All data (except crime index) comes from the FBI's "Table 8" for the year 2008, which is available on their website in various formats.

In the table, the cities are ordered by proximity to the searched location.

This website is a beta.

What do the marker colors represent?

  • red: high crime index
  • yellow: medium crime index
  • green: low crime index

These categorizations are relative to the section of the map that is zoomed to the searched city and surroundings. They are not absolute.
